🚨 Stage 1: Basic EC2 (Security Issues)

Stage 1 Architecture

Components

  • EC2 Instance: t2.micro with hardcoded AMI
  • Security Group: Ports 80, 22 open to 0.0.0.0/0
  • Apache Web Server: Displays demo page

⚠️ Security Issues Demonstrated

  • Hardcoded AMI ID (ami-0779c82fbb81e731c) - region-specific
  • Security group open to entire internet (0.0.0.0/0)
  • No configuration management
  • SSH port 22 exposed globally

Demo Steps

1. Deploy: echo "stage1-basic-ec2\napply" > demo-config.txt && git push
2. Show Issues: Point out hardcoded values and security vulnerabilities
3. Access: Visit http://<public-ip> to see the running instance

Components

  • Parameter Store: /demo/ec2/ami-id, /demo/ec2/instance-type
  • EC2 Instance: Dynamic configuration from parameters
  • Security Group: Configurable CIDR blocks
  • Data Sources: Terraform reads from Parameter Store

✅ Security Improvements

  • Dynamic AMI selection - works in any region
  • Externalized configuration management
  • Configurable security group rules
  • No hardcoded values in code
# Parameter Store Values (Auto-created) /demo/ec2/ami-id = "ami-0779c82fbb81e731c" /demo/ec2/instance-type = "t2.micro" /demo/security/allowed-cidr = "0.0.0.0/0"

🔐 Stage 3: Secrets Manager Integration

Stage 3 Architecture

AWS Secrets
Manager
→
Parameter
Store
→
EC2 Instance
Secure Config

Components

  • Secrets Manager: demo/app/api-key with encrypted credentials
  • Parameter Store: Non-sensitive configuration
  • IAM Permissions: Fine-grained secret access
  • Runtime Retrieval: Secrets fetched during deployment

🏆 Enterprise Security Features

  • AES-256 encryption at rest and in transit
  • Automatic rotation capability
  • CloudTrail audit logging
  • No secrets in Terraform state or code
  • Fine-grained IAM access control
Cost vs Security: Secrets Manager costs ~$0.40/month per secret, but prevents costly security breaches (average cost: $4.45M)

  • terraform plan -out=tfplan
  • terraform apply -auto-approve tfplan
↓

terraform output

  • terraform destroy -auto-approve
